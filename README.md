# Quai Antique

Quai Antique est une application web front-end destinée à un restaurant situé à Chambéry. Elle proposera une vitrine publique, un espace client, un espace d’administration et un parcours de réservation connecté à terme à une API Symfony.

## Technologies prévues

- HTML ;
- CSS et Sass ;
- Bootstrap ;
- JavaScript Vanilla ;
- routage JavaScript personnalisé ;
- `fetch` pour la future communication avec l’API Symfony ;
- Git pour le suivi des versions.

Le projet n’utilisera pas de framework JavaScript tel qu’Angular, React ou Vue.

## État actuel

Le projet se trouve dans sa phase initiale de documentation. Le besoin, les acteurs, les pages, les règles métier, le futur design system et la feuille de route sont définis avant le démarrage de l’implémentation technique.

## Grandes phases de réalisation

1. Analyse fonctionnelle et définition du périmètre.
2. Conception des parcours, des pages et de l’interface.
3. Initialisation de l’environnement front-end.
4. Développement de la partie publique responsive.
5. Développement des comptes et de l’espace client.
6. Développement de l’administration et des opérations CRUD.
7. Connexion à l’API Symfony et sécurisation des échanges.
8. Tests, corrections et mise en ligne.
